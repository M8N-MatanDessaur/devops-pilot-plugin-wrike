# Wrike Plugin for DevOps Pilot

Manage Wrike tasks, projects, and sprints with AI-powered actions.

## Installation

1. Clone this repo into your DevOps Pilot plugins folder:
   ```
   git clone https://github.com/M8N-MatanDessaur/devops-pilot-plugin-wrike.git dashboard/plugins/wrike
   ```
2. Restart DevOps Pilot
3. Go to Settings > Plugins > Wrike and add your access token

## Features

- Task list with folder filter, status filters, My Tasks toggle
- Board view with drag-and-drop status changes
- Task detail with comments, custom fields, approvals
- Cross-Sync modal (Wrike <-> Azure DevOps)
- Blueprint picker for creating from templates
- Searchable folder picker
- Command palette integration (Ctrl+K searches Wrike tasks)
- PS1 scripts for My Tasks and Standup summaries
- AI-powered actions via sidebar
